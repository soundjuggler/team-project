# Letterboxxed Movie Tracker

Letterboxxed Movie Tracker made by movie lovers for movie lovers.

This is a basic project built for a team project class, focusing on our collaboration and development workflow.

## About the Project

The goal of this project is to track and explore movies in a simple and user-friendly way. It was built as part of a team assignment to demonstrate collaboration, version control, and full-stack development skills.

## Features

- Movie tracking functionality
- Simple and clean interface
- API integration for movie data
- Team-based development workflow

## Setup

1. Clone the repository
2. Run `npm install`
3. Add required environment variables in your local config file
4. Start the server with `node server.js`
